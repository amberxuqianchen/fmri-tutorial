A step-by-step tutorial series guiding researchers through the full neuroimaging workflow in social neuroscience:

**Raw DICOM → BIDS → Quality Control → Preprocessing → First-Level GLM Ready**

Each module is a self-contained Jupyter notebook with narrative explanation, annotated code, and reproducible examples drawn from openly licensed datasets.

## Module Overview

| # | Notebook | Description |
|---|----------|-------------|
| 00 | `00_setup_check.ipynb` | Verify the Python environment and all required packages |
| 01 | `01_dicom_to_bids.ipynb` | Convert raw DICOM files to BIDS format using `dcm2bids` |
| 02 | `02_bids_validation.ipynb` | Validate a BIDS dataset with `bids-validator` and interpret common errors |
| 03 | `03_mriqc_quality_control.ipynb` | Run MRIQC and interpret image quality metrics (IQMs) |
| 04 | `04_fmriprep_preprocessing.ipynb` | Execute fMRIPrep and review its HTML reports |
| 05 | `05_confound_exploration.ipynb` | Explore and select nuisance regressors from fMRIPrep confounds TSV files |
| 06 | `06_first_level_glm.ipynb` | Build a first-level GLM with `nilearn`: design matrices and contrast maps |
| 07 | `07_roi_analysis.ipynb` | Extract region-of-interest (ROI) time series and run ROI-level statistics |
| 08 | `08_connectivity_analysis.ipynb` | Compute functional connectivity matrices and seed-based correlations |
| 09 | `09_group_level_analysis.ipynb` | Second-level (group) GLM: one-sample and two-sample t-tests |
| 10 | `10_visualisation_reporting.ipynb` | Publication-ready figure generation and results reporting |

## Prerequisites

| Requirement | Notes |
|-------------|-------|
| Python ≥ 3.9 | Tested on 3.9 – 3.12 |
| JupyterLab or Jupyter Notebook | Any recent version |
| Git ≥ 2.30 | For cloning and contributing |
| Docker (optional) | Required for MRIQC and fMRIPrep modules (04–05) |
| ~5 GB free disk space | For example datasets downloaded during notebooks |

Basic familiarity with Python, NumPy, and neuroimaging concepts (BOLD signal, HRF) is assumed. No prior experience with BIDS or any specific tool is required.

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/your-org/fmri-tutorial.git
cd fmri-tutorial
```

### 2. Create and activate a virtual environment

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Register the Jupyter kernel

```bash
python -m ipykernel install --user --name fmri-tutorial --display-name "fMRI Tutorial"
```

### 5. Launch JupyterLab

```bash
jupyter lab
```

Open `00_setup_check.ipynb` first to confirm your environment is configured correctly, then work through the modules in order.
